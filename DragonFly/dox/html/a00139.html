Opis szczegółowy

Klasa modeluje pojęcie Plaskowyz jako element powierzchni. Plaskowyz powstaje poprzez deformacje prostopadloscianu. Ta klasa jest klasą pochodną i jest ona specjalizacją klasy BrylaGeometryczna. Jej atrybutami są:

  • Polozenie, które określa położenie Bryły w trójwymiarze.

bool Plaskowyz::Tworz ( ) const
overridevirtual

Tworzy zapis wierzchołków bryły geometrycznej reprezentującej płaskowyż.

Warunek wstępny
Strumień wejściowy jest otwarty do odczytu i zawiera zapis współrzędnych wierzchołków wzorcowego prostapadłościanu. Zapis ich jest zgodny z formatem pozwalającym poprawnie zwizualizować prostopadłościan przez program gnuplot.
Zwracane wartości
true- plik z wierzcholkami nowej bryły reprezentującej płaskowyż został zapisany poprawnie.
false- operacja nie powiodła się.
