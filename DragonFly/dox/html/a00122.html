Opis szczegółowy

Klasa modeluje pojęcie Gory Z Dluga Grania jako element powierzchni. Gora Z Dluga Grania powstaje poprzez deformacje prostopadloscianu. Ta klasa jest klasą pochodną i jest ona specjalizacją klasy BrylaGeometryczna. Jej atrybutami są: Polozenie, które określa położenie Bryły w trójwymiarze.

bool GoraZDlugaGrania::Tworz ( ) const
overridevirtual

Tworzy zapis wierzchołków bryły geometrycznej reprezentującej górę z długą granią.

Warunek wstępny
Plik wzorcowy zawiera zapis współrzędnych wierzchołków wzorcowego prostapadłościanu. Zapis ich jest zgodny z formatem pozwalającym poprawnie zwizualizować prostopadłościan przez program gnuplot.
Zwracane wartości
true- plik z wierzcholkami nowej bryły reprezentującej górę z długą granią został zapisany poprawnie.
false- operacja nie powiodła się.
