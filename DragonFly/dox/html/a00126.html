Opis szczegółowy

Klasa modeluje pojęcie Gory Z Ostrym Szczytem jako element powierzchni. Gora z OStrym szczytem powstaje poprzez deformacje prostopadloscianu. Ta klasa jest klasą pochodną i jest ona specjalizacją klasy BrylaGeometryczna. Jej atrybutami są: Polozenie, które określa położenie Bryły w trójwymiarze.

◆ Tworz()

bool GoraZOstrymSzczytem::Tworz ( ) const
overridevirtual

Tworzy zapis wierzchołków bryły geometrycznej reprezentującej górę z ostrym wierzchołkiem.

Warunek wstępny
Plik wzorcowy zawiera zapis współrzędnych wierzchołków wzorcowego prostapadłościanu. Zapis ich jest zgodny z formatem pozwalającym poprawnie zwizualizować prostopadłościan przez program gnuplot.
Zwracane wartości
true- plik z wierzcholkami nowej bryły reprezentującej górę z ostrym szczytem został zapisany poprawnie.
false- operacja nie powiodła się.
